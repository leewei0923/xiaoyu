## 小鱼博客 (xiaoYu)

> 个人博客项目(新手小白) 2022-3-27
>
> React + Next.js (前台) , express(Node.js)(后台)



## 项目背景

从开始接触个人博客和学习前端技术就想着有一天，可以真正拥有自己写的个人博客。所以该博客，就是利用自己目前的技术栈，编写自己的博客。想实现什么样的功能和效果，努力去尝试，练手和学习两不误。因为是新手，还存在很多bug，后面时间慢慢修复bug和提供更多好玩的东西。

## 项目定位

- 个人博客
- 练手项目
- 持续更新，修复bug

## 功能

目前功能 🎨，  实验功能 🎈， 未来计划 🎁

- 主页
  - 混合与技术文章切换 🎨
  - header 标语🎨
  - 点击标签进入新的页面🎁
- 笔记、文档
  - 连载功能 🎨
    -  会有很多分类 学习笔记
  - 多导航 / 目录 (移动端也适配) 🎨
- 作品集
  - 展示个人作品 🎁
- 归档
  - 文章的目录 🎨
  - 增加标签 🎁
- 照片墙
  - 目前是点击进入图片灯箱,未来用来展示图片故事 🎁
- 留言录
  - 留言页面美化, 留言逻辑优化 🎁
- 关于作者
  - 展示个人资料 🎨
  - 展示博客的开发历程  🎨
  - 展示每日刷题的记录，类似github的功能 🎈
  - 美化页面 🎁

未来实现的功能 日历:

- [ ] 刷题展示页面
- [ ] 留言算法题目内容
- [ ] 日历展示(目前处于实验功能中，还有很多bug没解决）

## 技术栈

### 前端

- 开发框架: React、Next
- 组件库：Ant Design
- 语法扩展 ：Sass
- 打包工具：webpack

### 后端

- 主语言：JavaScript （node.js)
- 开发框架: Express
- 数据库: MongoDB
- 验证: JWT

### 部署

- web 服务：Nginx
- 图片存储: 七牛云
