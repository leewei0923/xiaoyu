## 安装 eslint

```shell
npm i eslint
```

eslint 初始化

```shell
npx eslint --init
```

eslint 修复

```shell
eslint 文件 . --fix
```

下载prettier 安装包

```shell
npm i -D prettier
```

prettier 格式化

```shell
npx prettier --write index.js
```
